# Gagner des gemmes

sur le serveur vous disposer actuellement de 4 manière différente pour obtenir des Gemmes:&#x20;

Pour chaque vote vous avez 15% de chance d'obtenir des gemmes ainsi que durant les vote party.

Le system de quêtes vous permet également de gagner des gemmes.



Au spawn vous trouverez un collectionneur qui vous propose d'échanger des item contre des gemmes. Il est également disponnible dans le nether.
